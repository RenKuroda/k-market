## 各画面の最低限UI要件

### `ログイン前トップ/簡易検索`

- **目的**: サービス理解＋簡易検索で在庫/相場感を提示し、登録/ログインに誘導する
- **表示**:
  - サービス概要（数行）
  - 簡易検索フォーム
  - 簡易検索結果一覧（サムネ・カテゴリ・メーカー/機種・所在地・価格目安）
- **入力**:
  - エリア（都道府県/市区町村）
  - カテゴリ（重機/ダンプ/アタッチ）
  - 提供形態（レンタル/売買/両方）
- **主要ボタン**:
  - 「検索」
  - 「会員登録/ログイン」
  - 一覧の「詳細を見る」（ログイン誘導）

### `認証（ログイン/会員登録）`

- **目的**: 企業ユーザーとしてログイン/登録できる状態にする
- **表示**:
  - ログインフォーム
  - 会員登録フォーム（最小項目）
- **入力**:
  - ログイン: email, password（Supabase Auth）
  - 登録: 会社名、company_type（需要/供給/両方）、担当者名、email, password、住所（都道府県/市区町村任意）
- **主要ボタン**:
  - 「ログイン」
  - 「新規登録」

### `需要側ダッシュボード（案件一覧）`

- **目的**: 見積〜案件の状況を一覧で把握し、次の行動（検索/詳細）に繋げる
- **表示**:
  - 案件一覧（見積中/発注済/完了）
  - 各行: 対象機械、期間、ステータス、供給側企業名（発注後）
- **入力**:
  - フィルタ（ステータス）
- **主要ボタン**:
  - 「新しい機械を探す」
  - 案件行クリック → `見積・案件詳細（需要側）`

### `機械検索/一覧（ログイン後）`

- **目的**: 条件検索で候補を絞り、詳細確認に繋げる
- **表示**:
  - 検索フォーム（エリア/カテゴリ/提供形態/利用期間）
  - 結果一覧（サムネ、メーカー/機種、所在地、提供形態、価格目安）
- **入力**:
  - エリア（必須）
  - カテゴリ（必須）
  - 提供形態（任意）
  - 利用期間（任意だが推奨）
- **主要ボタン**:
  - 「検索/再検索」
  - 「詳細を見る」→ `機械詳細`

### `機械詳細`

- **目的**: 見積依頼を出す判断材料を提供する
- **表示**:
  - スペック（カテゴリ/メーカー/機種/クラス/仕様）
  - 所在地
  - 提供形態（レンタル/売買）
  - 価格目安（レンタル日額/売買価格）
  - 写真
- **入力**: なし
- **主要ボタン**:
  - 「見積依頼する」→ `見積依頼作成`
  - 「一覧に戻る」

### `見積依頼作成`

- **目的**: 構造化された見積依頼を作成し送信する（1機械に対して）
- **表示**:
  - 対象機械サマリ（機械名/所在地/提供形態）
- **入力**:
  - 現場名（任意）
  - 現場住所（必須）
  - 利用期間（開始/終了、売買時は終了任意）
  - 台数（必須）
  - 搬入希望日（任意）、搬出希望日（任意）
  - 備考（任意）
- **主要ボタン**:
  - 「見積を依頼する」
  - 「キャンセル」

### `見積・案件詳細（需要側）`

- **目的**: 見積依頼内容・見積回答を確認し、発注確定する
- **表示**:
  - 依頼内容（現場/期間/台数/備考）
  - 見積回答（到着後: 金額・運搬費・条件メモ）
  - ステータス（見積中/発注済/完了）
- **入力**: なし（MVPでは再依頼/交渉UIは作らない）
- **主要ボタン**:
  - 「この内容で発注する」（回答あり、かつ未発注の場合）
  - 「一覧へ戻る」

### `供給側ダッシュボード（出品一覧＋見積/案件一覧）`

- **目的**: 出品管理と見積/案件対応の入口を統合し、画面数を減らす
- **表示**:
  - 出品一覧（機械名、提供形態、掲載ステータス）
  - 見積/案件一覧（相手企業、期間、ステータス）
- **入力**:
  - フィルタ（任意: ステータス）
- **主要ボタン**:
  - 「機械を登録する」→ `機械登録/編集`
  - 一覧行クリック → `見積・案件詳細（供給側）` / `機械登録/編集`

### `機械登録/編集`

- **目的**: 供給側が機械を登録し、検索対象として掲載できるようにする
- **表示**:
  - 既存値（編集時）
- **入力**:
  - カテゴリ、メーカー、機種名、クラス、仕様メモ
  - 提供形態（レンタル/売買/両方）
  - 価格（レンタル日額、売買価格）
  - 所在地（都道府県/市区町村）
  - 写真（最低1枚）
  - 掲載ステータス（掲載/停止の切替）
- **主要ボタン**:
  - 「保存」
  - 「戻る」

### `見積・案件詳細（供給側）`

- **目的**: 見積依頼を確認し、見積回答を作成・送信する
- **表示**:
  - 依頼内容（現場/期間/台数/備考）
  - 対象機械サマリ
  - 既存回答（回答済みの場合）
  - ステータス（見積中/発注済/完了）
- **入力**:
  - 見積回答（未回答時）: 基本金額、運搬費（任意）、条件メモ（任意）
- **主要ボタン**:
  - 「見積を送信する」（未回答時）
  - 「戻る」

### `運営ログイン`

- **目的**: 運営ロールで管理画面に入る
- **表示**: ログインフォーム
- **入力**: email, password（Supabase Auth）
- **主要ボタン**: 「ログイン」

### `運営ダッシュボード`

- **目的**: 企業/出品/案件管理への入口
- **表示**: リンク（企業/出品/案件）
- **入力**: なし
- **主要ボタン**:
  - 「企業管理」
  - 「出品管理」
  - 「案件一覧」

### `企業一覧+簡易詳細`

- **目的**: 企業の有効/無効を切替できる
- **表示**:
  - 企業一覧（会社名、company_type、status）
  - 選択時の簡易詳細（所在地など）
- **入力**: フィルタ（任意）
- **主要ボタン**: 「有効/無効切替」

### `出品一覧+簡易詳細`

- **目的**: 出品の掲載/停止を切替できる
- **表示**:
  - 出品一覧（機械名、会社名、status）
  - 選択時の簡易詳細（スペック/価格/所在地）
- **入力**: フィルタ（任意）
- **主要ボタン**: 「掲載/停止切替」

### `案件一覧+簡易詳細`

- **目的**: 案件を俯瞰し、必要ならステータスを手動更新する
- **表示**:
  - 案件一覧（相手企業、対象機械、status、期間）
  - 選択時の簡易詳細（見積/発注の内容）
- **入力**: フィルタ（任意）
- **主要ボタン**:
  - 「ステータス更新」（必要最小限）
